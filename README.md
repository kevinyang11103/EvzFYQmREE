# 船舶监造管理系统 java303

## 项目概述

船舶监造管理系统是基于Springboot和Vue的前后端分离项目，运用Javaee技术构建。本项目采用MVC三层体系结构设计，集成了Spring、SpringBoot、Mybatis及Vue等主流技术栈，通过Maven进行项目管理。系统使用MySQL数据库进行数据存储，实现了基于B/S架构的Web应用。

## 技术栈

- **后端：**
  - Spring
  - SpringBoot
  - Mybatis
- **前端：**
  - Vue
- **数据库：**
  - MySQL
- **项目管理：**
  - Maven

## 功能模块

- 船舶信息管理
- 监造任务分配
- 进度跟踪
- 质量控制
- 报告生成

## 系统角色

- 系统管理员
- 项目经理
- 监造工程师
- 质量检验员

## 运行环境

- **服务器：**
  - 推荐使用Java JDK 1.8 或以上版本
- **数据库：**
  - MySQL 5.7 或以上版本
- **浏览器：**
  - 支持现代Web标准的主流浏览器

## 部署步骤

1. 环境准备：安装Java JDK和MySQL数据库
2. 数据库搭建：执行SQL脚本建立数据库和表结构
3. 后端部署：编译后端代码，配置好数据库连接，部署到服务器
4. 前端部署：编译前端代码，配置好代理（如需），部署到Web服务器
